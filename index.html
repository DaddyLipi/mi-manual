Elemento Resumen
HTML Raiz del documento.
Head Contiene informacion que no esta a plena vista respecto al documento.
Title Este etiqueta va contenida en head y su funcionalidad es indicar el nombre que se muestra en la pagina del buscador.
Main Representa la informacion dominante dentro el body.
Div Es un contenedor generico que no tiene efecto en el contenido a menos que se le apliquen estilos.
P El elemento hace referencia a un parrafo, pueden ser vistos como bloques de texto.
Input El es usado para recibir informacion del usuario, este varia dependiento el type, por ejemplo button, date, password, etc.
H1-h6 Representan jerarquia en titulos siendo h1 la mas alta y h6 la mas baja.
Pan Es un inline element a diferencia de div es usado comunmente en texto.
Article Es un elemento que contiene informacion independiente de la pagina.
Section Es un elemento que se usa como contenedor para un area de la pagina.
Nav Se usa para agregar enlaces ya sea dentro el documento o hacia otro.
Footer Etiqueta que contiene informacion en la parte inferior de la pagina, esta puede ser informacion de la pagina como autor, links, copyright, etc.
Aside Se usa para presentar informacion que no esta muy relacionada al topico general, usualmente se presenta a los lados de la pagina.
Strong Sirve para indicar la importancia del contenido, usualmente el texto se renderiza en bold.
Sup Se usa para renderizar un texto un poco mas arriba que otro, por ejemplo un exponente.
Sub Se usa para renderizar un texto un poco mas abajo que otro.
Br Se usa para agregar un salto de linea.
Hr Se despliega una linea que se interpreta como cambio de tematica.
Wbr Es una etiqueta que sirve para indicarle al navegador que puede insertar un salto de linea.
Em Es una etiqueta que sirve para dar enfasis a un texto dado.
Mark Es una etiqueta que sirve para resaltar un texto.
Q Sirve para hacer una cita bibliografica referente al texto contenido.
Blockquote Sirve para hacer una cita bibliografica similar a la etiqueta "q" pero es desplegada como un cuadro.
Cite Sirve para hacer una cita pero se puede redireccionar a ella, esta debe tener un titulo.
Dfn Es usado para marcar la palabra que va a ser definida posteriormente.
Abbr Es usado para marcar que una cadena es una abreviacion o acronimo.
Small Se usa para desplegar el texto en un tama√±o de fuente mas peque√±o.
Ins Es usado para resaltar contenido que fue a√±adido recientemente al html.
Del Es usado para resaltar contenido que sera eliminado recientemente al html.
Code Se usa para desplegar un fragmento en formato de codigo, debe estar dentro el elemento pre, ya que code solo es una linea.
Pre Sirve para renderizar el fragmento interior en el mismo formato que fue ingresado.
Samp Es usado para marcar el texto contenido como un output de la maquina.
Kbd La etiquta despliega el texto contenido indicando que se espera una entrada de teclado.
Address Se usa para indicar que el contenido es una informacion de contacto.
Special Chars Es una secuencia de caracteres que permite ingresar caracteres como ser.
